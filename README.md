# Sqwizzey Switch

Индикатор раскладки клавиатуры для Windows 11. При переключении языка показывает по центру экрана полупрозрачную карточку с кодом языка (EN, RU и т.д.) — аналогично macOS.

## Что делает

- Всплывающий overlay при смене раскладки (Win+Space / Alt+Shift)
- Автоматически исчезает через 800 мс
- Не перехватывает фокус и клики мышью
- Не отображается в Alt+Tab
- Работает на всех мониторах с поддержкой разного DPI
- Настройки: позиция, прозрачность, время показа, тема (Dark / Light / Auto)
- Исключение полноэкранных приложений

## Стек

C# / .NET 8 / WPF / Win32 API
